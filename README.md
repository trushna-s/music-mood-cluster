# Music Mood Clustering with K-Means

K-Means clustering model to classify Spotify songs into mood clusters based on audio features.

## Dataset
- Source: [Spotify Classification Dataset](https://www.kaggle.com/datasets/geomack/spotifyclassification)

## Features Used
- Danceability
- Energy
- Loudness
- Speechiness
- Acousticness
- Instrumentalness
- Liveness
- Valence
- Tempo

## How to Run
1. Upload the code to Google Colab
2. Upload the `data.csv` f
3. View the clustering results and visualizations
